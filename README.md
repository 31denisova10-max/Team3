# RobodexBRICS — Киберфизическая платформа

> **Desert Horizon · Season I**
> Образовательная инженерная платформа в стиле Ori and the Blind Forest × Берлога

---

## О проекте

RobodexBRICS — браузерная образовательная платформа, где игроки проектируют роботизированные системы для реальных инженерных задач. Пилотный сезон разворачивается в пустыне Деште-Лут (Иран, 2050 год).

**Стек:** чистый HTML + CSS + Canvas API. Никаких зависимостей, никаких сборщиков — один файл.

---

## Геймплей

Миссия **«Дрон-инспектор К-7»**:
- Собери дрона из 9 слотов (корпус, навигация, батарея, сенсор, 4 двигателя, ИИ)
- Бюджет: 12 000 ¤
- Требования: термостойкость ≥ 60°C, автономная навигация без GPS, ≥ 4 часа работы
- Все компоненты должны быть сертифицированы по стандарту ISIRI
- Пройди испытание → получи сертификат Talent Bridge

## Технологии

- **Canvas API** — живые частицы света, органический пустынный фон
- **CSS Grid** — layout игрового экрана
- **CSS animations** — пропеллеры, LED-индикаторы, glow-эффекты в стиле Ори
- **Cinzel + Lato** — типографика (Google Fonts)
- **SVG** — дрон, карта мира, кластерные узлы, сигиль
